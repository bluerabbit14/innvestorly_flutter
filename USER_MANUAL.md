## üí∞ Revenue Tracking (Home Tab)

### **Understanding the Revenue Page**

The Revenue page shows your investment returns and hotel performance.

#### **Time Period Buttons**
At the top, you'll see four buttons:
- **Daily**: Previous day's revenue
- **Weekly**: Last 7 days revenue
- **Monthly**: Current year's monthly data
- **Yearly**: All historical yearly data

**How to Use**:
- Tap any button to switch views
- Data automatically refreshes when you switch

#### **Statistics Cards**

Two cards show key metrics:

1. **Total Gross Rev**
   - Sum of all hotel revenues for the selected period
   - Displayed in currency format (e.g., $50k, $1.2M)

2. **My Gross Share**
   - Your personal share of the total revenue
   - Based on your investment percentage

#### **Monthly View**

When you select **Monthly**:

1. **Month Dropdown**
   - Appears below the statistics cards
   - Select any month to view hotel-wise breakdown
   - Shows available months from the API

2. **Chart**
   - Bar chart showing revenue for each hotel
   - Two bars per hotel:
     - **Light Blue**: Total Gross Revenue
     - **Dark Blue**: Your Gross Share
   - Tap any bar to see detailed information

3. **Table**
   - Lists all hotels with:
     - Hotel name
     - Total Revenue
     - Investment Percentage (%)
     - Your Share amount

#### **Yearly View**

When you select **Yearly**:

1. **Hotel Dropdown**
   - Select "All Hotels" to see aggregated data
   - Or choose a specific hotel for detailed view

2. **Chart**
   - Shows yearly revenue trends
   - Compare different years side-by-side

3. **Table**
   - Year-wise breakdown
   - Total revenue and your share per year

#### **Daily & Weekly Views**

- Show hotel-wise revenue for the selected period
- Similar chart and table layout
- No dropdown filters needed

## üìà Year-over-Year (YoY) Reports

### **Accessing YoY Reports**

1. Go to **Reports** tab
2. Tap **"YoY Report"** card

### **Using YoY Reports**

#### **Year Pair Selection**

1. **First Dropdown**: Select year pair
   - Format: "2024-2025" (compares 2024 vs 2025)
   - Available pairs: 2024-2025, 2023-2024, 2022-2023

2. **Second Dropdown**: Select hotel filter
   - "All Hotel": Shows aggregated data
   - Specific hotel: Shows data for that hotel only

#### **Monthly Comparison Chart**

- Shows 12 months side-by-side
- Two bars per month:
  - **Light Blue**: Current year (e.g., 2025)
  - **Dark Grey**: Previous year (e.g., 2024)
- Compare month-by-month performance
- Tap bars for detailed revenue amounts

#### **Hotel Comparison Table**

Shows for each hotel:
- Hotel name
- Current year revenue (e.g., 2025)
- Previous year revenue (e.g., 2024)
- Change percentage (growth or decline)

**Understanding Change %**:
- **Positive %**: Revenue increased
- **Negative %**: Revenue decreased
- **N/A**: No data for previous year (new hotel)

## üîí Security & Authentication

### **Setting Up Authentication**

After logging in, you can add extra security:

1. Go to **Settings** tab
2. Tap **"Set Authentication"** card
3. Choose your preferred method:

#### **Option 1: Biometric Authentication**

**Requirements**:
- Device must support fingerprint or face recognition
- Biometric must be set up in device settings first

**Setup Steps**:
1. Tap **"Biometric Authentication"** card
2. Tap **"Continue"**
3. Authenticate using your device's biometric
4. Success! Biometric is now enabled

**Using Biometric**:
- App will prompt for biometric when you open it
- Quick and secure access
- Falls back to MPIN if biometric fails

#### **Option 2: MPIN (Mobile PIN)**

**Setup Steps**:
1. Tap **"MPIN Authentication"** card
2. Tap **"Continue"**
3. Enter a 4-digit PIN
4. Confirm the PIN
5. Success! MPIN is now set

**MPIN Rules**:
- Must be exactly 4 digits
- ‚ùå Cannot be sequential (e.g., 1234, 4321)
- ‚ùå Cannot be repeated (e.g., 1111, 2222)
- ‚ùå Cannot be common PINs (e.g., 0000, 2024)
- ‚úÖ Must be unique and secure

**Using MPIN**:
- App will prompt for MPIN when you open it
- Enter your 4-digit PIN to access

### **Removing Authentication**

1. Go to **Settings** ‚Üí **Set Authentication**
2. Select the authentication method you want to remove
3. Tap **"Remove [Method]"** button
4. Confirm removal

## üí° Tips & Best Practices

### **Data Refresh**
- Data is served from in-memory hardcoded data service
- Data automatically refreshes when you switch tabs
- Restart the app to reset all data to default values

### **Reading Charts**
- Tap on any bar in charts to see detailed information
- Tooltips show exact values
- Charts are interactive and responsive

### **Filtering Data**
- Use dropdowns to filter by month, year, or hotel
- "All Hotels" shows aggregated data
- Specific hotel selection shows detailed breakdown

### **Security Tips**
- Set up authentication for extra security
- Use a strong MPIN (not your birth year or common numbers)
- Keep your password secure
- Log out when using shared devices

### **Troubleshooting**

**Data Not Loading?**
- Restart the app to refresh data
- Try logging out and back in
- Verify you're using the correct login credentials

**Can't Login?**
- Verify phone number format (10 digits: `1234567890`)
- Check password (case-sensitive: `password123`)
- Ensure you're using the correct credentials

**Biometric Not Working?**
- Check device settings for biometric setup
- Try using MPIN instead
- Restart the app

**Theme Not Changing?**
- Toggle the switch again
- Restart the app
- Check if theme preference is saved

## üìå Important Notes

### **Hardcoded Data Mode**
This version of the app uses hardcoded data instead of API calls. All revenue, occupancy, and profile data is served from an in-memory data service. This means:
- No internet connection required
- Data persists only during the app session
- Profile updates are saved in-memory (phone number cannot be changed)
- Restarting the app resets profile changes to default values
- Revenue and occupancy data always returns the same hardcoded values
